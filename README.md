# 🐍 Snake Game

A classic **Snake game** built with **HTML, CSS, and JavaScript**! 🟩🟥  

Control a growing snake on a canvas, eat food, and avoid walls and your own tail. Simple, fun, and perfect for practicing JavaScript and canvas game logic.  

---

## 🎮 How to Play

- **⬆️ Up Arrow** – Move Up  
- **⬇️ Down Arrow** – Move Down  
- **⬅️ Left Arrow** – Move Left  
- **➡️ Right Arrow** – Move Right  

- Eat the **red blocks** 🍎 to grow your snake.  
- Avoid hitting the **walls** 🧱 or your **snake’s own body** 🟩🟩.  
- If the game ends 💀, you need to **refresh the browser** to restart.  

---

## ⚡ Features

- 🟢 Snake grows when it eats food.  
- 🟥 Food spawns randomly but **not on edges**.  
- ⬆️⬇️⬅️➡️ Smooth arrow key controls with 180° turn prevention.  
- 🕹️ Canvas-based game loop using `setInterval` for smooth movement.  

---

## 🛠️ Tech Stack

- **HTML5 Canvas** – Game board  
- **JavaScript** – Game logic & controls  
- **CSS** – Basic styling
